# Invoice Template Placeholder

Place your company's invoice Excel template here.

**File name**: `invoice-template.xlsx`

**Format**: Excel (.xlsx)

This template will be used as the master template for all generated invoices. The system will populate it with:
- Invoice number and date
- Customer details
- PO number and date
- Line items with quantities, rates, and taxes
- Totals including GST calculations
- Amount in words

## Template Requirements

Your template should maintain:
1. Company header with logo and details
2. GST registration information
3. Bank details (optional)
4. Professional formatting
5. Space for dynamic fields as per setup guide

See `docs/invoice-system-setup.md` for detailed cell reference requirements.
